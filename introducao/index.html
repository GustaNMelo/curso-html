O que são TAGS?

Em HTML, as tags são marcações utilizadas para estruturar e organizar o conteúdo de uma página na web. Elas funcionam como comandos que indicam ao navegador qual é a função de cada parte do texto ou elemento, como títulos, parágrafos, links ou imagens. Normalmente, uma tag abre para iniciar o conteúdo e outra fecha para indicar o seu fim, permitindo que tudo seja exibido de forma correta e compreensível para o usuário.

Elementos de Citação

Em HTML, os elementos de citação são usados para destacar trechos de texto que fazem referência a outras fontes ou que merecem uma formatação especial para indicar que não são parte do conteúdo principal. Eles permitem marcar citações diretas, citações curtas em linha, referências a títulos de obras ou até mesmo indicar a autoria de uma fala. Isso ajuda não só na organização visual da página, mas também na semântica, já que informa aos navegadores e mecanismos de busca que aquele trecho possui um significado específico relacionado a uma citação.
